Getting Started wit Arduino

//Install Nano BLE sense board in Arduino
Tools>Board manager>Search"nano BLE">Arduino nRF528x Boards(Mbed OS)

//Install libraries 
Sketch>Include libraries>Manage libraries>Search for"LSM9DS1" and "HTS221"

#include Arduino_LSM9DS1.h
// For reading accelerometer and gyroscope from Arduino nano BLE sense

#include Arduino_HTS221.h
// For reading temperature from Arduino nano BLE sense

#include BLE.h
// Bluetooth module for Arduino nano BLE sense
